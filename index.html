Instructions:
1. Press "Program"
2. Enter code:
    a. * to raise octave of note
    b. Note code number
    c. Rhythm code number
    d. Comma (fixed tune button)
3. "00" to end it

Notes:
C - 0
C#/Db - 1
D - 2
D#/Eb - 3
E - 4
F - 5
F#/Gb - 6
G - 7
G#/Ab - 8
A - 9
A#/Bb - A
B - b

Rhythms:
Eighth Note - 1
Quarter Note - 2
Dotted Quarter Note - 3
Half Note - 4
Dotted Half Note - 5
Whole Note - 6

Press program to enter the code for your song:
